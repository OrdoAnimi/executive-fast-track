# Design System Notes

## Source alignment

Executive Fast Track should visually inherit from Velocity Academy. It is a separate product stream but should feel like it belongs inside the same learning ecosystem.

## Core tokens

- Background: `#F8FAFC`.
- Alternate background: `#F1F5F9`.
- Surface: `#FFFFFF`.
- Text: `#0F172A`.
- Muted text: `#475569`.
- Border: `#E2E8F0`.
- Accent: `#E8630A`.
- Indigo: `#3730A3`.
- Navy: `#0F172A`.
- Body font: Inter-style system stack.
- Heading token: Playfair Display / Georgia reserved for future editorial sections, but current site uses the Velocity Academy weight system.

## Component language

Use:

- Sticky white navigation.
- Navy gradient hero.
- Orange hero bar.
- Orange primary CTA.
- White bordered cards.
- Small uppercase section labels.
- Dashboard-style course blocks.
- Dark band for Copilot and library-like material.
- Slate footer.

Avoid:

- Generic SaaS gradients.
- Bright course-marketplace styling.
- Excessive animation.
- Playful icons.
- Overdesigned dashboards.
- Dense text walls.

## Product personality

The site should feel serious, calm, executive, practical, and premium. It should not feel academic, childish, motivational, or overbuilt.

## UX rule

Every page or section must answer one executive question quickly:

- What is this?
- Who is it for?
- Why does it matter?
- What does it train?
- What does the learner produce?
