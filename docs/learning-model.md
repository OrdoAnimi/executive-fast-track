# Learning Model

## Name

Multimodal Immersion Loop.

## Principle

Executive Fast Track is video-first, not video-only. The course supports multiple learning modes because adult learners absorb material differently depending on energy, attention, context, and prior knowledge.

The course does not force a single pathway. It provides video, transcript, PDF notes, optional reading, cool notes, Copilot summaries, Feynman-style explanation, spaced recall, and scenario application.

## Learning loop

The standard module loop is:

1. Watch the video anchor.
2. Read or skim the transcript or sidecar PDF.
3. Capture rough notes or paste transcript excerpts.
4. Generate cool notes through the Copilot.
5. Extract executive vocabulary and terms to avoid.
6. Explain the concept simply using a Feynman-style replay.
7. Apply the idea in an executive scenario.
8. Review later through retrieval, not passive rereading.

## Learner modes

### Low-energy mode

Used when the learner has limited capacity but still wants progress. The learner watches or skims and asks for cool notes. No forced questioning, no quiz behaviour, no guilt loop.

Expected output:

- Cool notes.
- One boardroom sentence.
- Three terms to steal.
- One practical takeaway.

### Study mode

Used when the learner wants to consolidate material. The learner pastes rough notes, transcript fragments, or reading highlights. The Copilot structures the material.

Expected output:

- Executive summary.
- Concepts explained plainly.
- Vocabulary.
- Examples.
- Gaps or unclear points.
- Optional Feynman replay prompt.

### Performance mode

Used when the learner wants pressure and practice. The Copilot becomes an executive simulation partner.

Expected output:

- Role-specific objections.
- CEO, CFO, CIO, CMO, CISO, COO, CRO, and Chair framing.
- Stronger answer rewrites.
- Decision memo or pitch improvement.
- Boardroom-ready phrasing.

## Design rule

Skimming is allowed. Skimming is reconnaissance. The course should support first-pass recognition, second-pass understanding, third-pass translation, fourth-pass application, and later retention.

## Academy-wide standard

Video-first, transcript-supported, note-driven, Copilot-assisted, retrieval-reinforced, scenario-applied.
