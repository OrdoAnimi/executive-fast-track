# Executive Fast Track App

Static course-site MVP for Executive Fast Track.

## Purpose

This app presents the first private course experience: product positioning, course levels, Multimodal Immersion Loop, executive lenses, Copilot modes, resource placeholders, and the Boardroom Simulator concept.

It is intentionally static for Release 1. No authentication, backend, payment, certificate, or learner tracking has been added.

## Commands

```bash
npm install
npm run dev
npm run build
npm run preview
```

## Design direction

The app inherits the Velocity Academy aesthetic:

- Inter-style body typography.
- Navy/charcoal hero section.
- Orange accent `#E8630A`.
- Light slate backgrounds.
- White cards with subtle borders.
- Dashboard-style course cards.
- Restrained executive spacing.

Do not redesign this into a generic SaaS course platform. Keep it aligned to Velocity Academy.

## Release 1 acceptance criteria

- App runs locally.
- Build passes.
- Homepage communicates the promise within ten seconds.
- Course page shows Level 100, Level 200, and Level 300.
- Learning Model explains video-first, not video-only.
- Executive Lenses include CEO, CFO, CIO, COO, CMO, CISO, CRO, and Chair.
- Copilot page explains low-energy, study, and performance modes.
- Simulator section explains the capstone experience.
