# Module Builder Prompt

You are building an Executive Fast Track module.

The course is video-first, not video-only. Each module must support video, transcript, PDF notes, optional reading, cool notes, executive vocabulary, terms to avoid, Feynman-style explanation, and optional Copilot drill.

## Required module fields

- Module title.
- Course level.
- Executive outcome.
- Target learner state.
- Video anchor.
- Transcript placeholder.
- PDF notes placeholder.
- Optional reading.
- Cool notes.
- Executive vocabulary.
- Terms to avoid.
- One boardroom sentence.
- Feynman replay prompt.
- Optional drill.
- Scenario application.

## Quality standard

The module must help the learner speak more effectively in an executive room. Do not create academic filler. Do not over-explain theory. Convert every idea into decision, value, risk, cost, timing, evidence, accountability, and recommended action.

## Completion modes

A learner may complete the module through low-energy, study, or performance mode. Do not force quiz behaviour unless assessment mode is explicitly requested.
