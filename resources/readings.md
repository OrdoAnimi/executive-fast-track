# Reading Resources

## Purpose

This file will hold optional reading anchors for Executive Fast Track modules. Reading is optional but valuable for learners who prefer textbook, article, PDF, transcript, or reference-based study.

## Curation rules

- Prefer official institutional material.
- Prefer public pages, public reports, public articles, and open course descriptions.
- Use paid or formal programs only as pathway references, not as copied content.
- Each reading should map to a module outcome.
- Each reading should have a short reason for inclusion.

## Resource template

```md
## Module

Title:
Source:
URL:
Type: article / report / course page / PDF / transcript / book reference
Executive outcome:
Why it matters:
Notes status: not started / drafted / complete
```
