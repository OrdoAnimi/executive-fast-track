# Level 100 — Executive Language Foundations

## Purpose

Level 100 builds recognition and basic fluency. The learner starts hearing and using executive language across strategy, finance, market, risk, operations, governance, and decision-making.

This level is about pattern recognition, not mastery.

## Modules

1. Executive Thinking.
2. Strategy Without Architecture Jargon.
3. Finance for Non-Finance Leaders.
4. Boardroom Awareness.
5. Market Narrative and Commercial Positioning.
6. Risk, Security, and Assurance.
7. Operating Model and Execution Rhythm.

## Completion standard

The learner should be able to explain one complex idea without relying on technical, architecture, or framework-first language. The learner should be able to state the decision, value, risk, evidence, and recommended action in plain executive language.
