# Level 300 — Boardroom Simulation

## Purpose

Level 300 turns the course into performance practice. The learner brings a real idea, proposal, transformation, consulting offer, interview response, or product and tests it against a simulated executive room.

## Capstone pattern

1. Define the proposal.
2. Define the room.
3. State the decision required.
4. Explain the commercial value.
5. Name the risk and trade-off.
6. Answer executive objections.
7. Produce a boardroom-ready one-pager.
8. Produce a decision memo.

## Completion standard

The learner should be able to present a concise recommendation, defend it against role-specific executive objections, and leave with an artefact suitable for a senior conversation.
