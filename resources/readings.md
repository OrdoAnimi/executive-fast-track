# Reading Resources

## Purpose

This file holds optional reading anchors for Executive Fast Track modules. Reading is optional but valuable for learners who prefer textbook, article, PDF, transcript, or reference-based study.

## Curation rules

- Prefer official institutional material.
- Prefer public pages, public reports, public articles, and open course descriptions.
- Use paid or formal programs only as pathway references, not as copied content.
- Each reading should map to a module outcome.
- Each reading should have a short reason for inclusion.

## Level 100 reading anchors

### 100.1 — Executive Thinking

Title: Leadership and decision-making public executive education material.
Source: HBS Online, Stanford GSB, Yale SOM, MIT Sloan, or equivalent official public pages.
Type: public article / course page / video companion.
Executive outcome: Understand executive judgement, ambiguity, leadership decisions, and action under uncertainty.
Why it matters: This builds the learner's base mental model for executive communication.
