# Level 200 — Executive Lens Translation

## Purpose

Level 200 trains the learner to rotate the same idea through multiple executive lenses. The goal is not to become each executive role. The goal is to understand enough of each role's value system to frame recommendations in language they recognise as serious.

## Lenses

- CEO — growth, timing, strategy, execution confidence.
- CFO — capital, margin, cost, return, investment confidence.
- CIO — platform, integration, resilience, enterprise control.
- COO — operating rhythm, repeatability, execution, scale.
- CMO — market, narrative, demand, differentiation, trust.
- CISO — exposure, control maturity, incident readiness, evidence.
- CRO — risk appetite, assurance, accountability, regulatory posture.
- Chair and board — oversight, duty, trade-off, consequence, confidence.

## Completion standard

The learner should be able to take one proposal and produce role-specific framing for at least CEO, CFO, CIO, CMO, CISO/CRO, COO, and Chair.
